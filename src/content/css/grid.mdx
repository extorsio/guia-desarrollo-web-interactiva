## 7. Alineaci칩n en Grid

Grid tiene varias capas de alineaci칩n. Piensa en:

- Alineaci칩n dentro de cada celda (items).
- Alineaci칩n del contenido de la cuadr칤cula.
- Alineaci칩n de un item individual.

Propiedades clave:

- Para todos los items:
  - `justify-items`: alinea horizontalmente dentro de cada celda.
  - `align-items`: alinea verticalmente dentro de cada celda.
- Para el grid completo dentro de su contenedor:
  - `justify-content`: mueve el grid horizontalmente cuando sobra espacio.
  - `align-content`: mueve el grid verticalmente cuando sobra espacio.
- Para un item individual:
  - `justify-self`: alinea ese item horizontalmente.
  - `align-self`: alinea ese item verticalmente.

## 11. Mejores pr치cticas con CSS Grid

1. Usa Grid para layouts bidimensionales (filas + columnas). Para un eje 칰nico, Flexbox suele ser m치s simple.

4. Usa `grid-template-areas` en layouts complejos para mejorar legibilidad.

6. Usa `gap` en lugar de m치rgenes individuales entre items.
7. Aplica `grid-auto-flow: dense` solo cuando tenga sentido reordenar visualmente para rellenar huecos.
8. No abuses del posicionamiento manual l칤nea por l칤nea si el contenido es din치mico.
