# Selectores CSS

Los selectores CSS son patrones que nos permiten **seleccionar elementos HTML** para aplicarles estilos. Son una parte fundamental de CSS y entender cómo funcionan es esencial para crear diseños web efectivos.

## ¿Qué son los selectores?

Un selector CSS es la parte de una regla CSS que especifica qué elementos deben ser estilizados.

```css
selector {
  propiedad: valor;
}
```

Piensa en ellos como “filtros” para elegir qué elementos de tu HTML vas a pintar o modificar.

## Tipos de selectores básicos

### 1. Selector de tipo (Elemento)

Selecciona todos los elementos de un tipo específico.

<CodePlayground 
  title="Selector de elemento"
  html={`<p>Este es un párrafo</p>
<p>Este es otro párrafo</p>
<div>Este es un div (no será azul)</div>`}
  css={`p {
  color: blue;
  font-size: 18px;
  padding: 10px;
  background: lightblue;
  border-radius: 5px;
  margin: 5px 0;
}`}
/>

**Explicación:** Todos los elementos `<p>` (párrafos) tendrán el estilo aplicado.

---

### 2. Selector de clase (.)

Selecciona elementos que tienen un atributo `class` específico.

<CodePlayground 
  title="Selector de clase"
  html={`<p class="destacado">Este párrafo tiene la clase</p>
<p>Este párrafo NO tiene la clase</p>
<div class="destacado">Este div también tiene la clase</div>`}
  css={`.destacado {
  color: red;
  font-weight: bold;
  background: lightyellow;
  padding: 15px;
  border-left: 4px solid red;
  margin: 5px 0;
}`}
/>

**Uso:** Las clases se pueden reutilizar en múltiples elementos. Usa `.nombreClase` en CSS.

---

### 3. Selector de ID (#)

Selecciona un elemento único que tiene un `id` específico.

<CodePlayground 
  title="Selector de ID"
  html={`<div id="header">Este es el header principal</div>
<div>Este es otro div</div>`}
  css={`#header {
  background: #333;
  color: white;
  padding: 20px;
  text-align: center;
  font-size: 1.5rem;
  border-radius: 5px;
}`}
/>

**Importante:** Los IDs deben ser únicos en la página. Solo un elemento puede tener ese ID.

## Selectores combinados

### Selector de descendiente (espacio)

Selecciona elementos que son descendientes de otro elemento.

<CodePlayground 
  title="Selector descendiente"
  html={`<div class="container">
  <p>Este párrafo está dentro del container</p>
  <div>
    <p>Este también está dentro (descendiente indirecto)</p>
  </div>
</div>
<p>Este párrafo NO está dentro del container</p>`}
  css={`.container p {
  color: green;
  background: lightgreen;
  padding: 10px;
  border-radius: 5px;
  margin: 5px 0;
}

.container {
  border: 2px dashed gray;
  padding: 10px;
}`}
/>

**Explicación:** Selecciona todos los `<p>` que están dentro de `.container`, sin importar qué tan profundo.

---

### Selector de hijo directo (>)

Selecciona elementos que son hijos directos de otro elemento.

<CodePlayground 
  title="Hijo directo (>)"
  html={`<div class="padre">
  <p>Hijo directo (será verde)</p>
  <div>
    <p>Nieto (NO será verde)</p>
  </div>
</div>`}
  css={`.padre > p {
  color: white;
  background: green;
  padding: 15px;
  border-radius: 5px;
  font-weight: bold;
}

.padre {
  border: 2px solid green;
  padding: 10px;
}`}
/>

**Diferencia:** `>` solo selecciona hijos directos, no descendientes profundos.

---

### Selector de hermano adyacente (+)

Selecciona un elemento que sigue inmediatamente a otro.

<CodePlayground 
  title="Hermano adyacente (+)"
  html={`<h2>Título</h2>
<p>Este párrafo sigue inmediatamente al h2</p>
<p>Este NO sigue inmediatamente al h2</p>`}
  css={`h2 + p {
  color: red;
  font-size: 1.2rem;
  background: lightyellow;
  padding: 15px;
  border-left: 4px solid red;
}

h2 {
  color: #333;
  padding: 10px;
  background: lightgray;
  border-radius: 5px;
}`}
/>

**Uso:** Útil para estilizar el primer párrafo después de un título.

## Selectores de atributo

### Selector [atributo]

Selecciona elementos que tienen un atributo específico.

<CodePlayground 
  title="Selector de atributo"
  html={`<input type="text" placeholder="Campo de texto">
<input type="email" placeholder="Campo de email">
<input type="text" required placeholder="Campo requerido">`}
  css={`input[required] {
  border: 2px solid red;
  background: #ffe6e6;
  padding: 10px;
  border-radius: 5px;
  margin: 5px 0;
  display: block;
  width: 250px;
}

input {
  padding: 10px;
  border: 1px solid #ccc;
  border-radius: 5px;
  margin: 5px 0;
  display: block;
  width: 250px;
}`}
/>

---

### Selector [atributo="valor"]

Selecciona elementos con un valor de atributo exacto.

<CodePlayground 
  title="Atributo con valor exacto"
  html={`<input type="email" placeholder="Email aquí">
<input type="text" placeholder="Texto aquí">
<input type="password" placeholder="Contraseña aquí">`}
  css={`input[type="email"] {
  background: lightblue;
  border: 2px solid blue;
  padding: 10px;
  border-radius: 5px;
  margin: 5px 0;
  display: block;
  width: 250px;
}

input {
  padding: 10px;
  border: 1px solid #ccc;
  border-radius: 5px;
  margin: 5px 0;
  display: block;
  width: 250px;
}`}
/>

### :first-child y :last-child

<CodePlayground 
  title="Primer y último hijo"
  html={`<ul>
  <li>Primer elemento</li>
  <li>Elemento del medio</li>
  <li>Elemento del medio</li>
  <li>Último elemento</li>
</ul>`}
  css={`ul {
  list-style: none;
  padding: 0;
}

li {
  padding: 15px;
  background: lightgray;
  margin: 5px 0;
  border-radius: 5px;
}

li:first-child {
  background: lightgreen;
  font-weight: bold;
  border-left: 4px solid green;
}

li:last-child {
  background: lightcoral;
  font-weight: bold;
  border-left: 4px solid red;
}`}
/>

---

### :nth-child(n)

<CodePlayground 
  title=":nth-child - Selección por posición"
  html={`<div class="grid">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
  <div class="item">5</div>
  <div class="item">6</div>
</div>`}
  css={`.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

.item {
  background: lightblue;
  padding: 30px;
  text-align: center;
  border-radius: 5px;
  font-weight: bold;
}

.item:nth-child(odd) {
  /* Elementos impares (1, 3, 5) */
  background: lightyellow;
}

.item:nth-child(3) {
  /* Tercer elemento específicamente */
  background: lightcoral;
  color: white;
  font-size: 1.5rem;
}`}
/>

**Opciones:**
- `:nth-child(odd)`: Elementos impares
- `:nth-child(even)`: Elementos pares
- `:nth-child(3)`: Tercer elemento
- `:nth-child(3n)`: Cada 3 elementos

## Pseudo-elementos

### ::before y ::after

<CodePlayground 
  title="Pseudo-elementos ::before y ::after"
  html={`<p class="quote">Este es un texto con decoración</p>
<a href="#" class="external">Enlace externo</a>`}
  css={`.quote {
  position: relative;
  padding: 20px;
  background: #f0f0f0;
  border-radius: 5px;
  font-style: italic;
}

.quote::before {
  content: '"';
  font-size: 3rem;
  color: #3498db;
  position: absolute;
  left: 10px;
  top: 0;
}

.quote::after {
  content: '"';
  font-size: 3rem;
  color: #3498db;
  position: absolute;
  right: 10px;
  bottom: -10px;
}

.external {
  color: #3498db;
  text-decoration: none;
  font-size: 1.1rem;
}

.external::after {
  content: ' ↗';
  color: #e74c3c;
  font-weight: bold;
}`}
/>

**Uso:** Añaden contenido decorativo sin modificar el HTML.

## Combinando selectores

<CodePlayground 
  title="Selectores combinados avanzados"
  html={`<div class="card premium">
  <h3>Tarjeta Premium</h3>
  <p>Contenido especial</p>
</div>
<div class="card">
  <h3>Tarjeta Normal</h3>
  <p>Contenido normal</p>
</div>`}
  css={`.card {
  padding: 20px;
  margin: 10px 0;
  background: white;
  border: 1px solid #ddd;
  border-radius: 8px;
}

/* Selector de múltiples clases */
.card.premium {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  color: white;
  border: none;
  box-shadow: 0 10px 25px rgba(0,0,0,0.2);
}

.card h3 {
  margin-top: 0;
  font-size: 1.5rem;
}

.card.premium h3 {
  color: #fff;
  text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
}`}
/>

## Especificidad CSS

Cuando múltiples reglas se aplican al mismo elemento, CSS usa la **especificidad** para determinar cuál gana:

<CodePlayground 
  title="Especificidad en acción"
  html={`<p class="texto" id="especial">¿Qué color tendrá este texto?</p>`}
  css={`/* Especificidad: 0,0,0,1 */
p {
  color: blue;
}

/* Especificidad: 0,0,1,0 */
.texto {
  color: green;
}

/* Especificidad: 0,1,0,0 - ¡GANA! */
#especial {
  color: red;
  font-size: 1.5rem;
  font-weight: bold;
  padding: 15px;
  background: lightyellow;
  border-radius: 5px;
}`}
/>

**Orden de especificidad** (de mayor a menor):
1. **Inline styles** (`style="..."`)
2. **IDs** (`#id`)
3. **Clases, atributos, pseudo-clases** (`.class`, `[attr]`, `:hover`)
4. **Elementos** (`p`, `div`)

---

## Consejos y mejores prácticas

1. **Usa clases** en lugar de IDs para estilos (los IDs son muy específicos)
2. **Evita selectores muy profundos** como `.a .b .c .d .e`
3. **Nombres descriptivos** para las clases: `.btn-primary` mejor que `.azul`
4. **Mantén selectores simples** para mejor rendimiento
5. **Practica** con estos ejemplos interactivos
